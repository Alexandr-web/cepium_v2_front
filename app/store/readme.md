# Store в Nuxt 4 (Pinia)

Папка `app/store/` предназначена для глобального управления состоянием с помощью **Pinia** – официального решения для Vue. Nuxt 4 автоматически интегрирует Pinia и обеспечивает удобный автоимпорт хранилищ, поддержку SSR, а также бесшовную работу с композаблами и middleware.

## Основные концепции

- **Автоимпорт** – все файлы из `~/store/*.ts` автоматически доступны как composable `use*Store()` без явного импорта.
- **Реактивность** – хранилища Pinia построены на Vue-реактивности (`ref`, `computed`, `watch`).
- **SSR-ready** – состояния синхронизируются между сервером и клиентом, избегая утечек данных.
- **Глобальная доступность** – хранилища можно использовать в компонентах, composables, плагинах и middleware.
