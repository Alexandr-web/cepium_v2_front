# Utils - вспомогательные утилиты

Папка `app/utils/` содержит чистые функции-помощники, которые используются в разных частях проекта. Эти функции не имеют состояния, не работают с DOM и не зависят от конкретных фреймворков (например, Vue или Nuxt). Они предназначены для решения простых, повторяющихся задач: форматирование данных, математические расчёты, валидация, работа с датами и т.д.

## Назначение

- **Переиспользование** – утилиты вызываются из компонентов, composables, store, middleware.
- **Чистота** – функции детерминированы, не имеют побочных эффектов.
- **Тестируемость** – легко покрываются юнит-тестами.
- **Лёгкость** – не импортируют тяжёлые зависимости без необходимости.

## Структура папки

Рекомендуется группировать утилиты по темам или типам.
