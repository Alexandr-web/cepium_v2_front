# Types - TypeScript типы и интерфейсы

Папка `app/types/` содержит только декларации типов (интерфейсы, type aliases, enum) без реализации. Это единый источник истины для описания форм данных, которые используются в проекте: ответы API, пропсы компонентов, параметры функций, глобальное состояние и т.д.

## Назначение

- **Документация структуры данных** – быстро понять, как выглядит объект.
- **Типизация** – обеспечить строгую проверку типов во всём проекте.
- **Переиспользование** – одни и те же типы используются в моделях, компонентах, хранилищах и композаблах.
- **Разделение ответственности** – типы отделены от логики (классы в `app/models/` содержат методы, а `app/types/` – только описания).

Импортировать не надо, стоит автоимпорт.
