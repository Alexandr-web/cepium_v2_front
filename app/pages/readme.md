# Pages в Nuxt 4

Папка `app/pages/` - это основа маршрутизации в Nuxt. Каждый Vue-компонент в этой директории автоматически превращается в соответствующий URL-маршрут. Nuxt 4 использует улучшенную систему маршрутизации на базе `vue-router` с поддержкой динамических, вложенных и групповых маршрутов, а также предоставляет гибкие возможности для кастомизации через `router.options.ts`.

## Основные принципы

- **Файловая маршрутизация** – структура папок и файлов напрямую определяет пути URL[reference:0].
- **Автоматическое создание маршрутов** – Nuxt создаёт маршрут для каждого компонента в `~/pages/`[reference:1].
- **Гибкость** – поддерживаются динамические сегменты, вложенные и групповые маршруты.
- **Кастомизация** – полный контроль над маршрутизацией через `router.options.ts` и хуки[reference:2].
- **Опциональность** – если используется только `app.vue`, папка `pages/` не требуется, но при наличии `router.options.ts` система страниц активируется автоматически[reference:3].

## Использование

### Статические маршруты

| Файл | Маршрут |
|------|---------|
| `pages/index.vue` | `/` |
| `pages/about.vue` | `/about` |
| `pages/contact.vue` | `/contact` |

Для роутинга используем параметры в `router.options.ts`
