Данная папка содержит все изолированные функции и методы для выполнения HTTP-запросов к внешним и внутренним API-сервисам.

Также содержит папку `keys`, где указаны ключи для get-запросов tanstack

## 🚀 Основные правила разработки

* **Одна сущность — один файл:** Группируйте запросы по смыслу (например, все запросы к корзине — в `cart.ts`).
* **Только чистые функции:** Функции должны только отправлять запрос и возвращать данные или ошибку. Не управляйте состоянием UI (state) внутри этих файлов.
* **Типизация:** Если используется TypeScript, обязательно описывайте интерфейсы для `Request` и `Response` данных.
